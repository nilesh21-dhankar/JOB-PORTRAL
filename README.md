💼 Job Portal Web Application

A full-stack Job Portal web application where employers can post jobs and job seekers can apply, manage applications, and communicate efficiently.

---

🚀 Features

👤 Authentication

- User Signup & Login (JWT आधारित authentication)
- Secure password handling

🧑‍💼 Employer Features

- Post new job listings
- Manage (create/update/delete) job posts
- View applicants for each job

🧑‍🎓 Job Seeker Features

- Browse available jobs
- Apply to jobs
- View applied jobs history

💬 Messaging System

- Users can view messages/notifications
- Delete messages functionality

---

🛠️ Tech Stack

Frontend:

- React.js
- HTML, CSS, JavaScript

Backend:

- Node.js
- Express.js

Database:

- MongoDB

Authentication:

- JWT (JSON Web Token)
